# Overview

QueryPie 시스템의 공통 영역에서 발생하는 로그들을 General Logs에서 확인할 수 있습니다.

# QueryPie에서 제공하는 공통 로그

## User Access History
- 모든 사용자의 QueryPie 시스템 접근과 관련된 모든 이력, 즉 계정 로그인/아웃, 계정의 상태 변경, 비밀번호 재설정 메시지 전송 등의 이력을 조회할 수 있습니다.

## Activity Logs
- Activity logs에서는 관리자에 의하여 수행되는 리소스 등록 및 설정 변경 이력을 조회할 수 있습니다.

## Admin Role History
- QueryPie 관리자 권한의 부여 및 회수 이력을 조회할 수 있습니다.
- 관리자 권한 부여 및 회수는 **Administrator > General > User Management > Users** 메뉴에서 처리됩니다.

## Workflow Logs
- **QueryPie 결재 프로세스의 접수 및 승인 내역**을 조회할 수 있습니다.
- 사용자가 Workflow 페이지에서 특정 권한을 신청하거나 결재자가 승인 또는 반려하는 경우 Workflow Logs에서 이력을 확인할 수 있습니다.
