# MDM によるデバイスセキュリティの実現：Kandji x QueryPie

このウェビナーでは、当社のセキュリティ専門家であるブラント・ファン（QueryPie、CEO）とケニー・パーク（QueryPie、CISO）が、Apple デバイスを保護し、Kandji の強力な専用ツールを活用して、管理者とユーザーの両方に直感的で合理的なエクスペリエンスを提供する方法について説明します。

このセッションのモデレーターは、Shared IT の Jang Doohwan 様に務めていただきました。
