# CMS Renewal

Next.js 기반의 퍼블릭 웹사이트와 관리자 CMS를 함께 관리하는 프로젝트입니다.

## Overview

이 프로젝트는 다음 두 영역으로 구성됩니다.

- Public site
  - 다국어(`en`, `ko`, `ja`) 기반 퍼블릭 페이지
  - Demo / Documentation / News / Plans / Contact Us
- Admin CMS
  - 콘텐츠 생성, 수정, 임시저장, 게시 상태 관리
  - Demo / Documentation / News 관리
  - SEO 설정 관리

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Features

- 다국어 퍼블릭 페이지
- 관리자 콘텐츠 관리 UI
- Demo / Documentation / News 리스트 및 상세 흐름
- SEO 관리 메뉴
- 반응형 UI
- 공통 디자인 토큰 기반 스타일링
