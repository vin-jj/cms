Overview

QueryPie provides a web terminal for executing commands and a web SFTP for performing various tasks through a web browser.

Access via Web Terminal

STEP 1 Go to the Servers tab at the top and select the Role to use for access.

  • If no Role is assigned, the Default Role is automatically selected.

STEP 2 Select the server you need to access from the left, and choose the account from the right.

STEP 3 After selecting the protocol, click the Connect button.

Description

  • You can connect to multiple servers and switch between them seamlessly.

  • Click the + button on the top toolbar to open multiple sessions.

  • The web terminal supports a search function.
