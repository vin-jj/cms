🛠️ Core Demo Scenario (How it works)

1️⃣ Local File Access via Reverse Tunneling

With a single command, you activate QueryPie AIP's Edge Tunnel.
AIP securely connects to your local file path.

Product configurations and pricing data scattered across hundreds or even thousands of files
are analyzed instantly — without moving or uploading a single file.

2️⃣ Precise Analysis of Encrypted Secure PDFs

Even password-protected documents that cannot be opened manually are not a problem.
Using the password provided by the user, AIP securely accesses the file,
decrypts it, and analyzes the content — all without external exposure.

Complete security. Zero data leakage.

3️⃣ Secure File Retrieval via External Service Integration

AIP connects directly to external email services,
locates emails from specific senders, and retrieves attachments.

The files are then transferred into AIP's sandbox environment,
where they are analyzed safely within an isolated space.

「Check email → Download attachment → Verify password → Perform precise analysis」
The entire workflow is completed securely in one seamless flow.

Now, let AIP handle the complex procedures and repetitive tasks securely —
and you simply review the results.
