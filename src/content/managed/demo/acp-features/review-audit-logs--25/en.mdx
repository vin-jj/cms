# Overview

General Logs in QueryPie provide visibility into logs generated in the system's shared areas.

# Common Logs Provided by QueryPie

## User Access History
- You can review all records related to user access within the QueryPie system. This includes events such as account logins and logouts, changes to account status, and the transmission of password reset messages.

## Activity Logs
- Activity Logs allow you to monitor records of resource registrations and configuration changes that have been performed by administrators.

## Admin Role History
- The Admin Role History section enables you to track the history of admin role assignments and revocations.
- Admin role changes can be managed through **Administrator > General > User Management > Users**.

## Workflow Logs
- Workflow Logs provide logs related to **QueryPie’s workflow approval requests and approvals**.
- When users request specific permissions or when approvers approve or reject these requests on the Workflow page, the history of such actions is recorded in Workflow Logs.
