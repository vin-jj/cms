# Overview

QueryPie에서는 웹 브라우저를 통해 명령어를 실행할 수 있는 웹 터미널과 여러 작업을 수행할 수 있는 웹 SFTP를 제공합니다.

# 웹 터미널 사용하기

<BlueState>STEP 1</BlueState> 상단 **Servers** 탭으로 이동하여 접속을 위해 사용할 Role을 선택합니다.
- 부여된 Role이 없는 경우 Default Role이 자동으로 선택됩니다.

<BlueState>STEP 2</BlueState> 좌측에서 접속이 필요한 서버를 선택하고, 우측에서 계정을 선택합니다.

<BlueState>STEP 3</BlueState> Protocol을 선택한 후 `Connect` 버튼을 클릭합니다.

# 부가적인 편리 기능

- 다중 서버를 연결할 수 있으며, 서버 간 이동이 가능합니다.
- 상단 툴바의 `+` 버튼을 눌러 여러 세션을 열 수 있습니다.
- 터미널 내 검색 기능을 지원합니다.
