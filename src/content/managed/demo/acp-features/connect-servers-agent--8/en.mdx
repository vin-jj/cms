# Overview

Installing the QueryPie Agent allows you to use SSH clients like iTerm and SecureCRT. Users can connect to servers using QueryPie’s web Terminal or their existing tools with proxy credentials generated by QueryPie. This allows for controlled and secure access across various user environments, with logging and policy enforcement.

# Download the Agent App
<InfoNote>
    QueryPie Agent supports Mac, Windows, and Linux OS.
</InfoNote>

<BlueState>STEP 1</BlueState> After logging into QueryPie, click on your profile in the top right corner and select the `Agent Download` button.

<BlueState>STEP 2</BlueState> Download the installation file suitable for your PC's operating system.

# Connecting to a Server through the Agent
<InfoNote>
    **Precondition:** The useProxy option for the corresponding server must be enabled.
</InfoNote>

<BlueState>STEP 1</BlueState> Enter the QueryPie URL in the QueryPie Host input field and click the `Next` button.

<BlueState>STEP 2</BlueState> A web browser will open. Enter your credentials on the login page and click the `ontinue` button.

<BlueState>STEP 3</BlueState> Upon successful login, a login success screen will be displayed, and you will return to the Agent.

<BlueState>STEP 4</BlueState> If you have multiple roles, select a role before using the Server function after logging into the Agent.

<BlueState>STEP 5</BlueState> Right-click the server you want to connect to, select `Open Connection with`, and choose the terminal tool you want to use.

<BlueState>STEP 6</BlueState> Choose the desired account, enter the password if needed, and click the `OK` button to open a session.
