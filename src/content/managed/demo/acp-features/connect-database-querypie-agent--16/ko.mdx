# Overview

QueryPie Agent를 설치하면, DataGrip, DBeaver와 같은 3rd Party 애플리케이션을 사용할 수 있습니다. 웹 SQL 에디터와 함께 프록시 연결을 지원하여 다양한 사용자 환경에서도 문제없이 커넥션 접근을 제어하고, 정책을 적용할 수 있으며, 로그를 남길 수 있습니다.

# 에이전트 앱 다운로드

<InfoNote>
    QueryPie Agent는 Mac, Windows, Linux OS를 지원합니다.
</InfoNote>

<BlueState>STEP 1</BlueState> QueryPie 로그인 후 우측 상단 프로필을 클릭하여 `Agent Download` 버튼을 클릭합니다.

<BlueState>STEP 2</BlueState> PC 운영체제에 맞는 설치 파일을 다운로드합니다.

# 에이전트로 데이터베이스 접속하기

<InfoNote>
    **사전 조건:** DB 커넥션 메뉴에서 해당 커넥션의 Proxy Usage 옵션이 활성화되어야합니다.

    - **Use QueryPie Registered Account:** 관리자가 커넥션 정보 페이지 내에 저장한 DB username / password 기준으로 Proxy 접속 정보를 생성하는 방식입니다. 사용자는 Agent 또는 별도로 생성된 Proxy Credential 정보를 이용해 Proxy 로 접속할 수 있습니다.

    - **Use Existing Database Account with Agent:** 사용자가 기존에 사용하던 DB username / password 를 사용할 수 있는 방식으로 Proxy 접속 정보가 생성됩니다. 사용자는 Agent 에서 실행 후 localhost 와 port 정보를 이용해 Proxy 로 접속할 수 있습니다.
</InfoNote>

<BlueState>STEP 1</BlueState> QueryPie Host 입력란에 **QueryPie URL**을 입력하고 `Next` 버튼을 클릭합니다.

<BlueState>STEP 2</BlueState> 웹 브라우저가 열리면, 로그인 페이지에서 인증정보를 입력하고, `Continue` 버튼을 클릭합니다.

<BlueState>STEP 3</BlueState> 로그인을 성공하면 아래와 같이 로그인 성공 화면이 표시되며 이후 Agent로 돌아갑니다.

<BlueState>STEP 4</BlueState> Agent 앱 내 **Databases 탭**에서 권한 있는 커넥션들의 접속 정보를 확인할 수 있습니다.
- 접속할 커넥션에 할당된 `Port` 를 클릭하면, 해당 커넥션의 `Proxy Credentials` 정보를 확인할 수 있습니다.

<BlueState>STEP 5</BlueState> 접속 정보를 3rd Party 클라이언트에 입력하면 DB 커넥션 접속이 가능합니다.
