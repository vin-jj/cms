### 💡 SEO analysis, once considered the domain of specialists, can now be handled by an AIP agent.

This goes far beyond simply reading website text.
Analyzing a specific site's SEO (Search Engine Optimization), identifying issues that require immediate improvement, and transforming the results into an easy-to-understand visual dashboard —

These are tasks that would typically require hours of work from marketing professionals or data analysts.

However, QueryPie AIP completes this complex workflow in just minutes through the powerful combination of "Skills" and "Sandbox."

## 🛠️ Core Scenario of This Demo (How It Works)

In this video, we demonstrate how AIP performs intelligent actions using data from querypie.com.

**1️⃣ Advanced Analytical Logic: "Skill"**

When a user requests,
"Analyze the SEO performance and summarize improvement areas,"

AIP invokes a pre-built SEO analysis Skill.

This Skill goes beyond simple text generation. It parses real web data and calculates technical SEO metrics through structured analytical logic.

**2️⃣ Secure and Powerful Execution Environment: "Sandbox"**

The analyzed data is transferred to AIP's isolated and secure execution environment — the Sandbox.

Within the Sandbox, AIP processes the collected data and instantly generates a visualization dashboard complete with charts and graphs.

Instead of receiving a plain list of text outputs, users obtain a polished, presentation-ready result.

Furthermore, the HTML code generated within the Sandbox can be downloaded directly or integrated into internal dashboards with ease.
