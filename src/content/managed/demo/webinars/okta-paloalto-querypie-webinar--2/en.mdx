# The OPQ Webinar
As the sophistication of technology grows, so does the complexity of hackers. 
To create new security standards,OktaInc, PaloAlto Networks and QueryPie have joined forces to form the OPQ Alliance. 
Learn the best way to implement ISMS certifications, stay compliant and keep enterprise data safe, along with the implementation of Zero Trust Network, a security management system to streamline device security. 
Going beyond cloud security to cloud-native security, the OPQ Alliance (Okta x Palo Alto Networks x QueryPie) will show you the way. 
Watch this three-part webinar to learn more!
