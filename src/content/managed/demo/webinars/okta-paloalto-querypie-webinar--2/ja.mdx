# OPQ ウェビナー
技術が高度化するにつれて、ハッカーの攻撃方法もまた精巧になり、複雑になっています。

このような状況の中で、新しいセキュリティ標準を設けるために、Okta、Palo Alto Networks、そしてQueryPieが力を合わせてOPQアライアンスを結成しました。

ISMS認証の実装とコンプライアンスの維持、そして企業データを安全に保護する最適な方法をご紹介します。 また、機器のセキュリティをシンプル化するセキュリティ管理システムであるゼロトラストネットワーク(Zero Trust Network)の導入により、クラウドセキュリティを越えてクラウドネイティブセキュリティを実現する方法をご案内します。

OPQアライアンス(Okta x Palo Alto Networks x Query Pie)による3部作のウェビナーを視聴し、より詳しい内容を確認してください！
