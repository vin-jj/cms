# MDM을 통한 디바이스 보안의 완성: Kandji x QueryPie

이 웨비나에서는 보안 전문가인 Brant Hwang(쿼리파이 CEO)과 Kenny Park(쿼리파이 CISO)이 Apple 기기를 안전하게 보호하는 방법과, Kandji의 강력하고 목적에 맞게 설계된 도구를 활용해 관리자와 사용자 모두를 위한 직관적이고 간소화된 경험을 만드는 방법에 대해 논의합니다.

본 세션은 Shared IT의 장두환 님이 진행했습니다.
