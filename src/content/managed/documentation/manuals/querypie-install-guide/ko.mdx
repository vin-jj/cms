# 🧭 시작하기 전에
**QueryPie ACP Community 버전**은 **QueryPie Enterprise**에서 제공하는 핵심 기능을 체험해 볼 수 있는 버전입니다.
- Database Access Controller
- System Access Controller
- Kubernetes Access Controller
- Web Access Controller
<br />

단, Community 버전은 **최대 5명의 사용자 계정이 활성화 가능**합니다.

(5명 이상 사용자 등록이 필요하다면, <Link href="/plans" markdown external>Enterprise 플랜</Link>으로 확장해보세요.)

QueryPie ACP는 일반적인 Web Application 방식으로 작동하며, Proxy 기반의 Network Server 기능도 포함되어 있습니다.

# 🐳 설치 방법
설치는 Docker Image 기반이며, 한 줄 명령어로 자동 설치가 가능합니다.

## 1. 터미널에서 명령 실행하기

Linux 서버의 터미널에 접속한 뒤, 홈 디렉토리에서 아래 명령을 입력하세요:
```code
bash <(curl https://dl.querypie.com/setup.v2.sh)
```
⏱️ 설치에는 약 **7~10분** 정도 소요됩니다.

## 2. 설치되는 동안 라이선스 발급 받기

QueryPie ACP Community Edition은 라이선스 등록이 필요합니다.
설치 되는 동안 <Link href="/querypie/license/community/apply" markdown external>라이선스 발급 신청서</Link>를 통해 신청하면, 입력한 이메일로 .crt 형식의 텍스트 파일이 발송됩니다.

## 3. 설치 후 접속하기
설치가 완료되면 브라우저에서 다음 주소로 접속합니다:
```code
http://<서버 IP 주소>
또는
https://<서버 IP 주소>
```
설치한 리눅스 서버의 IP 주소는 이용자의 PC 에서 접근 가능한 IP 주소를 사용하여야 합니다.

## 4. 라이선스 등록하기
이메일로 받은 .crt 파일을 업로드하거나 PEM 형식 텍스트 내용을 복사/붙여넣기 하세요.

# 🔑 라이선스에 대한 안내
- QueryPie ACP를 사용하려면, 제품 설치 후 별도로 제공되는 라이선스를 등록하여야 합니다.
- 라이선스는 .crt 확장자의 텍스트 파일 형태로 제공됩니다.
- 라이선스는 신청 시 입력한 이메일 주소로 발송됩니다.
- 발급되는 라이선스는 신청일로부터 1년간 유효합니다.
- 라이선스는 신청자 개인에게 발급되며, 제3자에게 양도할 수 없습니다.
