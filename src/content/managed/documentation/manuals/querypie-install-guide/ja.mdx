# 🧭 始める前に
**QueryPie ACP Community Edition**は、**QueryPie Enterprise**で提供される主要な機能を体験できるバージョンです。
- データベースアクセスコントローラー
- システムアクセスコントローラー
- Kubernetesアクセスコントローラー
- Webアクセスコントローラー
<br />

ただし、Communityバージョンではユーザーの登録数に制限はありませんが、同時にアクティベートできるユーザーは**最大5人まで**となります。

(5名以上のユーザー登録が必要な場合は、<Link href="/plans" markdown external>Enterpriseプランへ</Link>のアップグレードをご検討ください。)

QueryPie ACPは一般的なWebアプリケーションとして動作し、プロキシベースのネットワークサーバー機能も含まれています。

# 🐳 インストール方法
インストールはDockerイメージを基にしており、ワンライナーコマンドで自動的に行えます。

## 1. ターミナルでコマンドを実行

Linuxサーバーのターミナルにアクセスし、ホームディレクトリで以下のコマンドを入力してください：
```code
bash <(curl https://dl.querypie.com/setup.v2.sh)
```
⏱️ インストールには約**7～10分**かかります。

## 2. インストール中にライセンスを取得

QueryPie ACP Community Editionのご利用にはライセンス登録が必要です。
インストール中に <Link href="/querypie/license/community/apply" markdown external>ライセンス申請フォーム</Link> から申請すると、入力したメールアドレス宛に .crt 形式のテキストファイルが送信されます。

## 3. インストール後にアクセス
インストールが完了したら、ブラウザで以下のアドレスにアクセスします：
```code
http://<サーバーIPアドレス>
または
https://<サーバーIPアドレス>
```
インストールしたLinuxサーバーのIPアドレスは、ユーザーのPCからアクセス可能なIPアドレスを使用する必要があります。

## 4. ライセンスを登録
メールで受け取った.crtファイルをアップロードするか、PEM形式のテキスト内容をコピー＆ペーストしてください。

# 🔑 ライセンスについて
- QueryPie ACPを利用するには、製品インストール後に別途提供されるライセンスを登録する必要があります。
- ライセンスは.crt拡張子のテキストファイルとして提供されます。
- ライセンスは申請時に入力したメールアドレスに送信されます。
- 発行されたライセンスは申請日から1年間有効です。
- ライセンスは申請者個人に発行され、第三者に譲渡することはできません。
